Installation

Extension Installer

  1. Uploading

    Go to Extensions > Extension Installer and upload file installer/sysinfo.ocmod.zip.

    NOTE If you previously have System Information installed, you will see replace file list. Proceed to replace them.

    NOTE If you get warning "using the same ID code as the one you are trying to upload!" mean you installing same version. Click the clear button and no need to proceed.

  2. Pre-installation

    1. Go to Extensions > Modifications, refresh modification to generate new modification cache.

      NOTE If you see modification with same name, delete the one with lower version before refresh modification.

    2. Go to System > Users > User Groups, open your user group. Check Access and Modify permission for System Information.

  3. Installation

    1. Find SI icon on left menu, click to open System Information.

Manual Install

If you have issues using Extension Installer, proceed with this manual installation.

  1. Upload Files

    Use FTP software and upload manual/upload to your site root.

    NOTE Do not upload the folder upload/, but folders inside it.

  2. Upload OcMod

    Go to Extensions > Extension Installer and upload file manual/sysinfo.ocmod.xml.

    NOTE If you get warning "using the same ID code as the one you are trying to upload!" mean you installing same version. Click the clear button and no need to proceed.

  3. Installation

    Next installation step is the same as Extension Installer: Pre-installation and Installation.
